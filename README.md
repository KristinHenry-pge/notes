## Installing node.js on centOS

1. Install EPEL, first 

`$ sudo yum install epel-release`


2. Get the source

`$ sudo curl --silent --location https://rpm.nodesource.com/setup_6.x | bash -`

3. Then, intall nodejs

`$ sudo yum -y install nodejs`
